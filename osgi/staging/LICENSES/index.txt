This directory reproduces the copying/licensing files from all
dependencies.  They are summarized below.

dsinfo, dsprofile: LGPL v3
algebra, cats, scallop, spire: MIT
shlurd, guava, evo-inflector, ixa-pipes: Apache 2
jgrapht: LGPL v2.1 or EPL
kiama: Mozilla 2
scala, extjwnl: 3-clause BSD
WordNet:  2-clause BSD

Note that extjwnl (including all WordNet 3.1 data) is repackaged into
shlurd-extjwnl-bundle_2.13.assembly.jar in order for resources to
be correctly loadable via OSGi.
