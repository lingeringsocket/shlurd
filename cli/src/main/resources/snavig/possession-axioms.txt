Import "/snavig/basic-axioms.txt".

A character-inventory must be a repository.
A stuff is a character-inventory.
A game-character must have a stuff.
The player-inventory is a repository.
The player-inventory is the player-character's stuff.

If a game-character gets a game-object, then
 the game-object must be in the game-character's container;
 otherwise the game-interpreter complains "But there is no such object here.".
If a game-character gets a game-object, then
 the game-object is in the game-character's stuff.
If a game-character drops a game-object, then
 the game-object must be in the game-character's stuff;
 otherwise the game-interpreter complains "But you have no such object.".
If a game-character drops a game-object, then
 the game-object is in the game-character's container.

If a game-character is holding a game-object, then equivalently
 the game-object is in the game-character's stuff.

If the game-turn advances, then
 the player-character perceives the player-inventory's spc-contained-objects.
